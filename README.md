#  To-Do List App (Go)

A simple **command-line To-Do List application** written in Go (Golang).  
You can easily add, view, and delete your daily tasks directly from the terminal.

##  Features
- Add new tasks  
- View all tasks  
- Delete completed tasks  
- Simple and clean CLI interface
